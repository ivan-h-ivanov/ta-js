# TA-JS Project

This project visualizes a candlestick chart using HTML5 Canvas and JavaScript.

## Getting Started

1. Clone the repository:
   ```sh
   git clone https://github.com/yourusername/ta-js.git
   cd ta-js
   ```

2. Open `root.html` in your browser to view the chart.
